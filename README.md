# Atlantic Hotel - Página de Reservas

Este repositorio contiene el código HTML y CSS para la página del Atlantic Hotel. La página está diseñada para permitir a los usuarios descubrir información sobre el hotel y realizar reservas de habitaciones en el hotel de manera fácil y rápida.

## Contenido

El repositorio incluye los siguientes archivos:

- `index.html`: El archivo HTML que define la estructura y el contenido de la página de reservas.
- `styles.css`: El archivo CSS que proporciona los estilos visuales y de diseño para la página.

## Estructura del Código

La página  incluye las siguientes secciones:

### Pagina principal

- **Encabezado**: Contiene el logotipo del hotel y enlaces a las redes sociales.
- **Menú de Navegación**: Permite a los usuarios navegar fácilmente por las diferentes secciones de la página.
- **Sección de Servicios**: Proporciona información sobre los servicios ofrecidos por el hotel, como spa, piscina, restaurante, bar y gimnasio.
- **Sección de Habitaciones**: Muestra las diferentes opciones de habitaciones disponibles en el hotel, incluyendo habitaciones presidenciales, normales y de lujo.
- **Sección de Contacto**: Permite a los usuarios ponerse en contacto con el hotel a través de un formulario y muestra la ubicación del hotel en un mapa de Google.
- **Pie de Página**: Contiene información de contacto del hotel y enlaces a las redes sociales.

### Reservsas

- **Encabezado**: Incluye el logotipo del hotel y enlaces a las redes sociales como Instagram, Twitter y Facebook.
- **Formulario de Reservas**: Permite a los usuarios ingresar su información personal, seleccionar fechas de llegada y salida, así como elegir el tipo de habitación que desean reservar.
- **Sección de Videos**: Muestra dos videos de YouTube incrustados que ofrecen información adicional sobre el hotel o sus servicios.
- **Pie de Página**: Contiene información de contacto del hotel y enlaces a las redes sociales.

## Personalización y Responsividad

El código HTML y CSS se puede personalizar fácilmente para adaptarse a las necesidades específicas del Atlantic Hotel. Los estilos pueden modificarse para cambiar los colores, las fuentes y el diseño de la página. Además, la página es completamente receptiva y se adapta automáticamente a diferentes tamaños de pantalla, garantizando una experiencia de usuario consistente en dispositivos móviles y de escritorio.

## Uso

Para utilizar la página de reservas en un proyecto, simplemente copia los archivos `index.html` y `styles.css` en el directorio de tu proyecto y enlaza el archivo CSS al archivo HTML. Luego, podrás personalizar el contenido y los estilos según tus necesidades.
